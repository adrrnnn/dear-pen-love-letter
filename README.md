# Love Letter Envelope

A responsive, animated envelope + letter you can customize (no build step).

## Run locally
- Open `index.html` in your browser.

## Customize
Edit `config.js`:
- `recipientName`
- `message` (array of paragraphs)
- `signature`

## Publish
You can host this as a static site on:
- GitHub Pages
- Netlify
- Vercel

## Structure
- `index.html` – markup
- `style.css` – styling + animations
- `script.js` – interaction logic
- `config.js` – customization
